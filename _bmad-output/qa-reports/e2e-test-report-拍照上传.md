#### 问题1: ocrService静态方法调用失败
**影响测试**: 4个测试
**错误信息**: `TypeError: Cannot read properties of undefined (reading 'recognizeQuestionType')`

**失败的测试**:
- 端到端集成测试 › 应该完成完整的用户旅程
- 边界条件 › 应该处理空字符串输入
- 边界条件 › 应该处理只有数字没有关键词的情况
- 边界条件 › 应该处理特殊字符输入

**根本原因**: `ocrService` 是类，`recognizeQuestionType` 是静态方法，但在测试中mock没有正确设置

**修复建议**:
```typescript
// 修改测试文件中的mock方式
jest.mock('../../services/ocrService', () => ({
  OCRService: {
    recognizeQuestionType: jest.fn().mockResolvedValue({
      questionType: QuestionType.ADDITION,
      confidence: 0.95,
      keywords: ['+'],
    }),
  },
}));
```
