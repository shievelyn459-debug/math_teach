# 功能名称：用户登录

> **填写日期**: 2026-03-25
> **填写人**: PM团队
> **优先级**: P0

---

## 功能描述
家长使用邮箱和密码登录App，进入孩子的学习管理界面

## 用户角色
家长

## 操作步骤
1. 打开App，进入登录页面
2. 输入注册邮箱
3. 输入密码
4. 点击"登录"按钮

## 期望结果
- 登录成功，进入首页
- 显示欢迎消息和用户名
- 显示已添加的孩子列表

## 相关代码位置
- 登录页面：`src/screens/LoginScreen.tsx`
- 登录服务：`src/services/authService.ts`
